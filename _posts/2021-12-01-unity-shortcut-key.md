middle 拖动

right 观察

W A S D + right 移动

F 视野平移并缩放到当前选中对象

shift 加速移动

---

shift + middle 平移

ctrl + middle 缩放

alt + middle 吸附旋转

0 切换视角

1 前视图

ctrl + 1 后视图

3 右视图

7 顶视图

5 投影切换

9 翻转

. 聚焦选中物体

/ 聚焦选中物体，屏蔽其他物体

shift + left 多选

ctrl + left 连选或少选

ctrl + right 套索选择

N 侧栏

T 工具栏

G + X Y Z + (shift) 移动 

S + X Y Z + (shift) 缩放

R + X Y Z + (shift) 旋转

alt + G S R 返回默认

shift + A 添加

shift + Z 着色方式切换

shift + D 复制

alt + D 链接复制

tab 编辑模式

ctrl + P 设置层级

ctrl + shift + R 编辑模式下一变三

ctrl + M 镜像对称

ctrl + G 合并

J 连接顶点路径

ctrl + + 邻接选择

shift + left 微调

ctrl + alt + Q 四视图

A 全选

A + A 取消全选

X 删除

ctrl + alt + 0 将当前视角设为相机视角

ctrl + space 窗口最大化
